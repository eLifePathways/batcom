## Getting Started

### Accessing the Platform

1. Open your web browser and navigate to the Bat-Com Research Platform URL
2. The home page displays an overview of the research initiative
3. Use the navigation menu at the top to explore different sections

### Creating an Account

To access administrative features:

1. Contact the system administrator to create an account
2. Once your account is created, click the "Log In" button in the top navigation
3. Enter your username and password
4. You will be redirected to the admin dashboard upon successful login

## Content Management

### Managing Virus Categories

To add a new virus category:

1. Go to Admin → Virus Categories
2. Click "Add New Category"
3. Fill out the form with:
   - Category name
   - Description
   - Image (optional)
4. Click "Save" to create the category

To edit an existing category:

1. Find the category in the list
2. Click the "Edit" button
3. Make your changes
4. Click "Save"

To delete a category:

1. Find the category in the list
2. Click the "Delete" button
3. Confirm deletion when prompted

### Managing Team Members

To add a new team member:

1. Go to Admin → Team Members
2. Click "Add Team Member"
3. Fill out the form with:
   - Name
   - Title
   - Institution
   - Description
   - Image
   - Contact information (optional)
4. Click "Save" to add the team member

To reorder team members:

1. Go to Admin → Team Members
2. Use the drag handles to reorder the list
3. Click "Save Order" to apply changes

### Managing Publications

To add a new publication:

1. Make sure you have setup "GraphQL API" under Admin settings
2. Go to Admin → Publications
3. Click "Refresh Publication"

### Managing Background Papers

To add a new background paper:

1. Go to Admin → Background Papers
2. Click "Add Background Paper"
3. Fill out the form with:
   - Title
   - Virus category
   - Link
   - Image (optional)
   - Description (optional)
4. Click "Save" to add the paper

### Managing "What We Do" Content

To add a new section:

1. Go to Admin → What We Do
2. Click "Add Section"
3. Fill out the form with:
   - Title
   - Subtitle (optional)
   - Description
   - Image (optional)
   - Slug (URL-friendly name)
4. Click "Save" to create the section

To add content to a section:

1. Go to Admin → What We Do
2. Find the section and click "Manage Content"
3. Click "Add Content"
4. Choose the content type (text, image, video, etc.)
5. Fill out the form with the appropriate content
6. Click "Save"

To reorder content:

1. Go to section content management
2. Use the drag handles to reorder the list
3. Click "Save Order" to apply changes

## Issue Reporting

### Reporting an Issue

For website administrators and researchers:

1. Click the bug icon in the bottom right corner of any page
2. Fill out the form with:
   - Issue title
   - Description
   - Email (optional)
   - Screenshot (automatically captured)
3. Click "Submit" to report the issue

### Managing Issues

To view and manage reported issues:

1. Go to Admin → Issues
2. View the list of all reported issues
3. Click on an issue to see detailed information
4. Update the status or priority as needed
5. Add comments to track progress

## Analytics Dashboard

To view site analytics:

1. Go to Admin → Analytics
2. View statistics about:
   - Visitor counts
   - Page views
   - Traffic sources
   - Device distribution
   - Popular pages
   - Geographic distribution

## Frequently Asked Questions

### General Questions

**Q: How do I request access to the admin panel?**  
A: Contact the system administrator at admin@batcom.org to request access.

**Q: How often is content updated?**  
A: The research team updates content regularly as new findings are published.

**Q: Can I download publications directly from the website?**  
A: Most publications provide links to the original sources where you can download the full papers.

### Technical Questions

**Q: What browsers are supported?**  
A: The platform supports all modern browsers including Chrome, Firefox, Safari, and Edge.

**Q: Is the platform mobile-friendly?**  
A: Yes, the website is fully responsive and can be accessed on mobile devices.

**Q: How do I report technical issues?**  
A: Use the bug reporting tool accessible from the icon in the bottom right corner of any page.

### Content Questions

**Q: How do I suggest a publication to be added?**  
A: Contact the research team through the Contact form with details about the publication.

**Q: Can I contribute content to the platform?**  
A: Researchers interested in contributing should contact the team lead through the Contact form.
